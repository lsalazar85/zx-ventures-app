# Zx Ventures App

Web app developed in Next.js


## Run Project

## Frontend

### Npm
```
    npm install
    npm start
```

### Yarn
```
    yarn install
    yarn start
```

## Development Technologies
* Modern Javascript ( ES6 )
* Next.js
* React.js
* React Hooks
* React Context State
* React Testing Library & Jest
* Styled-Components
* Lodash

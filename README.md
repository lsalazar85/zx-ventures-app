# Zx Ventures App

Web app developed in Next.js


## Run Project

## Frontend
Starts on `http://localhost:3000`

### Npm
```
    npm install
    npm start
```


### Yarn
```
    yarn install
    yarn start
```

## Api - Server
Starts on `http://localhost:5000`

### Npm
```
    npm start
```

## Development Technologies
* Modern Javascript ( ES6 )
* Next.js
* React.js
* React Hooks
* React Context State
* React Testing Library & Jest
* Styled-Components
* Lodash
* JSON Server
